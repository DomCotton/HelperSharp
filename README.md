HelperSharp
=========

A set of helpers and extensions for C# programming.

Setup
=========

NuGet
------
PM> Install-Package HelperSharp

Using
=========
Helpers
------
- CurrencyHelper


Extensions
------
- ArrayExtensions
- CharExtensions
- ConvertibleExtensions
- DateTimeExtensions
- EnumerableExtensions
- StringExtensions
